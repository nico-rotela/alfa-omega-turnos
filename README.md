# 💈 Alfa & Omega - Sistema de Turnos

Aplicación web desarrollada para la gestión de turnos de una barbería.

El objetivo del proyecto fue crear una agenda moderna, simple e intuitiva para administrar reservas, visualizar disponibilidad y gestionar turnos de clientes.

---

## ✨ Características

- Agenda visual por horarios.
- Creación de turnos.
- Edición de turnos existentes.
- Eliminación de turnos.
- Navegación por fechas.
- Visualización de horarios disponibles y reservados.
- Diseño responsive (Mobile First).
- Interfaz personalizada para la barbería Alfa & Omega.

## 🚀 Instalación

### Clonar repositorio

```bash
git clone <url-del-repo>
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Variables de entorno

Crear un archivo `.env` dentro de la carpeta backend:

```env
DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASSWORD=
```

## 📌 Funcionalidades principales

### Agenda visual

Permite visualizar rápidamente los horarios disponibles y reservados para cada día.

### Gestión de turnos

- Crear turnos.
- Editar turnos.
- Eliminar turnos.

### Organización por fecha

Los turnos se muestran agrupados por día para facilitar la administración de la agenda.
